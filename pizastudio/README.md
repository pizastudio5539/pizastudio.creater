# Piza Studio - ホームページ

クリエイティブスタジオ「Piza Studio」のモダンでレスポンシブなホームページです。

## 🚀 特徴

- **レスポンシブデザイン**: デスクトップ、タブレット、モバイルに対応
- **モダンなUI/UX**: 美しいグラデーションとアニメーション効果
- **インタラクティブ**: スムーズなスクロールとホバー効果
- **アクセシブル**: キーボードナビゲーションとタッチ操作に対応
- **パフォーマンス最適化**: 高速読み込みとスムーズな動作

## 📁 ファイル構成

```
pizastudio/
├── index.html          # メインHTMLファイル
├── styles.css          # スタイルシート
├── script.js           # JavaScript機能
└── README.md           # このファイル
```

## 🛠️ 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: Flexbox、Grid、アニメーション、メディアクエリ
- **JavaScript (ES6+)**: モダンなJavaScript機能
- **Font Awesome**: アイコンフォント
- **Google Fonts**: Noto Sans JP フォント

## 📱 対応デバイス

- デスクトップ (1200px以上)
- タブレット (768px - 1199px)
- モバイル (767px以下)

## 🎨 セクション構成

1. **ヒーローセクション**: メインビジュアルとCTA
2. **私たちについて**: 会社のビジョンと統計情報
3. **サービス**: 提供サービスの紹介
4. **ポートフォリオ**: 過去の作品紹介
5. **お問い合わせ**: 連絡フォームと連絡先情報
6. **フッター**: サイトマップとソーシャルリンク

## 🚀 使用方法

1. ファイルをWebサーバーにアップロード
2. `index.html`をブラウザで開く
3. 必要に応じて内容をカスタマイズ

## 🎯 カスタマイズ方法

### 色の変更
`styles.css`の以下の変数を変更：
```css
/* メインカラー */
#6366f1  /* プライマリカラー */
#8b5cf6  /* セカンダリカラー */
```

### コンテンツの変更
`index.html`の各セクションのテキストを編集

### アニメーションの調整
`script.js`のアニメーション設定を変更
